# Column Space

## 一句话理解

Column Space 是矩阵所有列向量张成的空间，也就是所有可能的 $Ax$ 输出集合。

## 相关问题

- columns span $\mathbb{R}^m$ 与 $Ax=b$ 对任意 $b$ 有解是什么关系？
- pivot row / pivot column 分别说明什么？
- Column Space 如何连接到 rank、consistency 和 linear independence？
