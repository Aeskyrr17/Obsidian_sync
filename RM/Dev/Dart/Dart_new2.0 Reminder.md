#### 力传感器
1. 用DMA+RS485。。。好难调
2. 如果dma_receive过快，容易把时序弄乱。变送器的频率比较低。
	-  给 USART2/3 加了一个“等待回复”标志
	- 只有不在等待上一次回复时才再次发起 HAL_UART_Transmit_DMA()
	- 收到 RxEvent 或 Error 后会清掉等待标志
	- 如果超过 20 tick 还没等到回复，会自动解锁并重启 ReceiveToIdle_DMA
