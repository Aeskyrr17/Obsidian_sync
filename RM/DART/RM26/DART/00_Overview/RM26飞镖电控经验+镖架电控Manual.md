# 写在前面

相对于其他机器人来说，飞镖是一个非常独立且特殊的兵种。它拥有着相对来说最稳定的工况，最固定的流程和完全自动的逻辑。

同时，由于兵种相对独立，镖架代码可能很难从同队伍的其他兵种处得到直接的参考，尤其包括一些特殊的元器件与电机，较为复杂机电系统的稳定协同。
