# 偏微分

diff(表达式, 变量)

- 对于一组变量，常用jacobian

```matlab
q   = [x; theta_l; theta_b];
dq  = [dx; dtheta_l; dtheta_b];
ddq = [ddx; ddtheta_l; ddtheta_b];
dLag_dq  = jacobian(Lag, q).';      <- jacobian默认是row vector，用.'转换为列向量
dLag_ddq = jacobian(Lag, dq).';
```

解方程技巧
```matlab
unknowns = [
    ddx
    ddtheta_l;
    ddtheta_b;
    f;
    F_w_h;
    F_w_v;
    F_l_h;
    F_l_v  ];

%八个牛顿欧拉方程
eqs_NE = [
    eq_b_x;
    eq_b_y;
    eq_b_rot;
    eq_l_x;
    eq_l_y;
    eq_l_rot;
    eq_w_x;
    eq_w_rot   ];

% 转换为 A * unknowns = b

[A_NE, b_NE] = equationsToMatrix(eqs_NE, unknowns);

% 联立求解
sol_NE = linsolve(A_NE, b_NE);

% 只对需要的结果进行化简，避免 simplify 太慢
ddx_expr = simplify(sol_NE(1), 'Steps', 50);
ddtheta_l_expr = simplify(sol_NE(2), 'Steps', 50);
ddtheta_b_expr = simplify(sol_NE(3), 'Steps', 50);
```
